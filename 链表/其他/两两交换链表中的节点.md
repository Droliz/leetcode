# 两两交换链表中的节点


## 题解

交换节点可以认为是交换节点的值

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
	def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
		if not head: return head
	
		cur = head
		flag = True
		while cur.next:
			if flag:
				cur.val, cur.next.val = cur.next.val, cur.val
				flag = False
			else:
				flag = True
			cur = cur.next
		return head
```

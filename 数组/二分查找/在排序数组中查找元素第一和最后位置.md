# 在排序数组中查找元素第一和最后位置

![[Pasted image 20221209234334.png]]

## 二分法

查找左边界和有边界即可

**左边界**

```python
left, right = 0, len(nums) - 1

while left <= right:
	mid = (left + right) >> 1
	if nums[mid] < target:
		left = mid + 1
	else:
		right = mid
return left
```

**右边界**

```python
left, right = 0, len(nums) - 1

while left < right:
	mid = (left + right + 1) >> 1
	if nums[mid] > target:
		right = mid - 1
	else:
		left = mid
return left
```


```python
class Solution:
	def searchRange(self, nums: List[int], target: int) -> List[int]:
		if target not in nums:
			return [-1, -1]
	
		# 找左边界
		def cur_left():
			left, right = 0, len(nums) - 1
	
			while left < right:
				mid = (left + right) >> 1
				if nums[mid] < target:
					left = mid + 1
				else:
					right = mid
			return left
	
		# 找右边界
		def cur_right():
			left, right = 0, len(nums) - 1
	
			while left < right:
				mid = (left + right + 1) >> 1
				if nums[mid] > target:
					right = mid - 1
				else:
					left = mid
			return left
	
		l = cur_left()
		r = cur_right()
	
		return [l, r]
```

# 两个数组的交集

![](Pasted%20image%2020221213213020.png)


## 题解

哈希表

```python
class Solution:
	def intersection(self, nums1: List[int], nums2: List[int]) -> List[int]:
		dic = {}
	
		for i in nums1:
			dic[i] = dic.get(i, False)
	
		for i in nums2:
			if dic.get(i) is not None:
				dic[i] = True
		return [i for i in dic.keys() if dic[i]]
```

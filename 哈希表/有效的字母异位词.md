## 题解

此题有很多解法，例如字典序排序、集合、哈希表

此处使用哈希表

```python
class Solution:
	def isAnagram(self, s: str, t: str) -> bool:
	
		dic = {}
	
		for i in s:
			dic[i] = dic.get(i, 0) + 1
	
		for i in t:
			if dic.get(i) is None:
				return False
			else:
				dic[i] -= 1
	
		res = 0
		for i in dic.values():
			res |= i
	
		return res == 0
```
